Lista los skills disponibles en Phi y cómo usarlos. Responde en español con formato claro.

Skills disponibles:
- `/status` — resumen del estado actual del proyecto (oportunidades, tareas, research)
- `/dossier [entidad]` — genera análisis de una oportunidad (ej: `/dossier NYCEDC`)
- `/outreach [entidad]` — estrategia de contacto con borrador de mensaje
- `/research [tema]` — investigación profunda multi-búsqueda (sub-agente)
- `/compare [entidad1] [entidad2]` — compara dos oportunidades del portafolio
- `/help` — esta ayuda

Puedes usar comillas para entidades con espacios: `/dossier "DOE Industrial Technologies"`
